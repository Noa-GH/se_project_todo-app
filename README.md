# Simple Todo App

This is a To-Do List App that's a TriplTen project to gain hands-on experience using OOP and optimization practices.

## Functionality

The main functionality of this project was to create a persistent SPA that can do basic ToDo List features like adding and removing of list items, checkmarking, and writing out inputs. The page is designed in a way with OOP to be able to prevent the browser from defaulting and reloading on submission and always allowing data to stay loaded on after submission or reset the form if inputs were cancelled.

## Technology

Tech used in this project: JavaScript, ES6, Git, and AI tools like; Claude & Co-Pilot.
